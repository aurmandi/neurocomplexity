# Figures for the Frontiers in Neuroinformatics paper

Five figures are planned. Generated by the package's `viz/` module
(matplotlib, Nature-style rcParams in `viz/_style.py`; SVG + PDF + 600
dpi TIFF triplet via `save_publication`).

| # | File | Source | Status |
|---|------|--------|--------|
| 1 | `fig1_overview.{svg,pdf,tif}` | `nc.viz.figure_overview(results)` — 6-panel composite of the example session | Pending end-to-end Allen run |
| 2 | `fig2_benchmark_matrix.{svg,pdf,tif}` | Built from `results/benchmarks/v1.1.0.csv` — case × tolerance heatmap | Awaits v1.1.0 CSV regeneration (in progress) |
| 3 | `fig3_branching_per_area.{svg,pdf,tif}` | `nc.viz.figure_branching` per cortical area, with bootstrap CIs | Pending Allen run |
| 4 | `fig4_te_connectivity.{svg,pdf,tif}` | Directed TE matrix across V1/LM/AL/RL/AM/PM, surrogate-null gated | Pending Allen run |
| 5 | `fig5_pid_summary.{svg,pdf,tif}` | `nc.viz.figure_pid` for selected triplets | Pending Allen run |

All five are referenced from `paper.md` with `![](figures/figN_*.pdf)`
and Frontiers-style numbered captions. The captions are drafted inline
in `paper.md`; only the underlying images need to be generated.
